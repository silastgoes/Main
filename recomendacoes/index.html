Curso de Git/Gitihub(Curso em Video)

Git e GitHub[20 Horas]: grátis, prático e sem usar comandos no terminal. Perfeito para iniciantes em programação que querem um lugar para guardar seus primeiros projetos. O curso conta com aulas teóricas de funcinamento e história das ferramnetas, sita concorrentes e tipos de software de vercionamento, além de aulas práticas utilizando a interface GithubDesktop. O aluno pode entrar no curso sem nuncater tido contato com as ferramentas, pois o curso ensina desde o cadastro a hospedagem de sites, como este site em questão.
